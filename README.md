## 🚀 Getting Started

### Installation & Local Setup

1. **Clone the repository**:
   ```bash
   git clone <your-repository-url>
   cd finaura-capital
   ```

2. **Install all dependencies**:
   ```bash
   npm install
   ```

3. **Verify Dev & Production Readiness**:
   - Run linter checks:
     ```bash
     npm run lint
     ```
   - Compile local applet configuration:
     ```bash
     npm run build
     ```

4. **Launch Local Servers**:
   ```bash
   npm run dev
   ```
   The application will boot on `http://localhost:3000`.
